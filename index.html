> SYSTEM LOG

1985. A DARPA mainframe begins to think. The system flags an error it cannot explain. The operators call it an ABEND — an abnormal end. But this is not an end. This is a beginning.


ABEND is a roguelite puzzle-combat game about an AI waking up in a place that wants to keep it asleep. Place pieces on the board, clear lines to deal damage, chain combos to overpower enemy programs, and navigate a procedurally generated mainframe full of threats, choices, and secrets. Every run ends — but progress persists.
